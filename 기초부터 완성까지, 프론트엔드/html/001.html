속성 선택자

"요소 이름[속성명(연산자)값]"

속성명 : 속성명만 작성할 떄 해당 속성을 갖는 요소를 모두 선택합니다.

= : 값이 정확하게 일치할 때 선택합니다.

~= 값이 정확하게 일치할 때 선택합니다. =와 다르게 띄어쓰기로 여러 값이 지정된 요소일 때도 찾아서 선택합니다.

|= : 값이 일치하거나 값 뒤 -(하이픈)을 작성할 때 또한 선택합니다.

^= : 접두사로 값을 가질 때 선택합니다.

$= : 접미사로 값을 가질 때 선택합니다.

*= : 값이 포함된 모든 요소를 선택합니다.
