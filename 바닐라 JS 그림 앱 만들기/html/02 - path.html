const canvas = document.querySelector("canvas");

const ctx = canvas.getContext("2d");

canvas.width = 200;

canvas.height = 200;

ctx.rect(10, 10, 50, 50);

ctx.stroke(); or ctx.fill


rect로 선을 만들어놓고, 한 번에 채우기도 가능하다.

ctx.rect(10, 10, 50, 50);
ctx.rect(20, 20, 60, 60);
ctx.rect(30, 30, 70, 70);
ctx.fill()
ctx.rect(40, 40, 80, 80);

이와 같이 작성한다면 맨 마지막 선은 그려지지 않고 앞선 3개의 선만 채워지게 된다.


(새경로 만들기) 기존의 선과는 다른 선을 만들기

…
ctx.rect(10,20,30,30)

ctx.beginPath();
ctx.rect(50,50,90,90);
ctx.storkestyle = "red";
ctx.storke();

beginPath()를 사용하면 구분선(끊어치기)처럼 뒤에 선들이 새로운 경로로 생성된다.

또한 canvas로 그린 그림들은 원한다면 경로를 나눌 수 있다.


Shortcut

ctx.fillRect(10,10,50,50);

앞 내용과 더불어 fillRect는 fill과 rect의 합본이다.

마찬가지로 storke도 가능함. 두 함수를 합쳤기에 따로 rect로 선을 그리고

그 선을 fill 할 필요 없이 처음부터 선과 칠을 동시에 할 수 있게 만듦
