#01 canvas

const canvas = document.querySelector("canvas");

⇢ canvas 태그를 JS에 변수 지정한다.

const ctx = canvas.getContext("2d");

⇢ context는 브러쉬이며 2d와 3d(webgl, webgl2, bitmaprenderer)중에 그릴 수 있다.

>canvas.height = 200;
pcanvas.width = 200;

⇢ 이미지 퀄리티를 위해서 js에서도 높이, 너비를 지정해준다.

ctx.fillRect(10, 10, 50, 50);

⇢ 4개의 argument를 가지고 있음. x, y, width와 height
canvas의 시작은 ↖(왼쪽 위) 코너에서부터 0, 0이다.
